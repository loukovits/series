# SLim base project
Slim base project with bootstrap code.

functionality:

    login/register
  
    Administration of users
  
    Permitions with midleware
  
    Full logging of requests
  
    Csrf protection
  
    Validations
    
    Flush messages
  
    Database Support with Eloquent ORM
  
    Twig views
    
    Migrations for database Schema creation
